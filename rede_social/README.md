# TCC - Web Scrapping

Nesse projeto, será explorada a extração, carregamento, transformação (ETL-Extraçao,Transformation and Loading) de dados por meiorda tecnologia de Web Scraping. 
O Objetivo geral é extrair de plataformas populares, como Instagram, Facebook e Twitter, informações de tags específicas e fornercer análises sobre essas informações.
